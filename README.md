# HR Analytics Dashboard

## Overview
This Power BI dashboard provides insights into employee attrition, workforce demographics, salary distribution, and department-wise employee analysis.

## Key Metrics
- Total Employees: 1403
- Active Employees: 1176
- Attrition Count: 227
- Attrition Rate: 16.18%
- Average Age: 36.97 Years
- Average Experience: 7.05 Years

## Features
- Employee Attrition Analysis
- Department-wise Employee Distribution
- Salary Slab Analysis
- Gender Distribution
- Age Group Analysis
- Experience Trend Analysis
- Interactive Filters and Visualizations

## Tools Used
- Power BI
- DAX
- Data Modeling
- Data Visualization
